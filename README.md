梶原健司の個人プロジェクト置き場。仕事とは関係なく、純粋に作りたいものを作る場所。

## このリポジトリについて

ずっと作りたいものがあった。でも技術者ではないから、一人では作れなかった。

Claude Codeが出てきたことで、それが初めてできるようになった。
1〜2年前にAIが登場した頃は、チャットでコピペするしかなく、エラーのたびに手が止まっていた。
今は短い時間でここまで動くものが作れる。この変化を自分自身が体験できていることが、このリポジトリの出発点でもある。

---

## 作りたいものの背景

技術は手段。目的は**人間の意識の拡張**だと思っている。

デジタル技術の本質的なすごさは、専門的なリテラシーがなくても人間の能力を拡張できること。
これからの時代、人間の認識や意識はもっと広がっていく。

今まで「見えなかったもの」を可視化することで、人は何かを新しく意識できるようになる。
たとえば地球全体に広がる笑顔を一つの画面で見られたら、地球を「一つのもの」として感じられるかもしれない。
そしてそれは、なるべくポジティブな方向に向いていてほしい。

できたものは、無料でいろんな人に見てもらえるようにしたい。
